# 玄上 Seedance 2.0 API 视频生成器

> 一个纯前端的多模态 AI 视频生成 Web UI，支持 Seedance 2.0 模型、素材库管理、字幕擦除。

---

## 功能特性

- **多模态参考生视频** — 支持图片、视频、音频作为参考素材，提示词编辑器支持 `@` 引用已上传素材
- **双模型支持** — Seedance 2.0 / Seedance 2.0 Fast，支持 480p / 720p、多种画面比例、4–15 秒时长
- **本地文件自动托管** — 拖拽上传的本地文件自动转存至 Litterbox / Uguu / tflink，获取公网可访问 URL
- **字幕擦除** — 对已完成的生成视频一键提交精细化字幕/水印擦除任务（Ark AMK）
- **智能任务轮询** — 自动并发轮询任务状态，实时显示进度、耗时与结果预览
- **历史记录管理** — 底部网格展示历史任务，支持结果预览、重新提交、字幕擦除、一键下载
- **主题与布局** — 亮色 / 暗色 / 绿色三种主题，左右侧栏可折叠、可拖拽调整宽度
- **纯前端零依赖** — 单文件 `index.html`，无需构建，直接浏览器打开即可使用

## 快速开始

1. 克隆或下载本项目
2. 直接用浏览器打开 `index.html`
3. 在「设置」面板中配置：
   - **API Key** — 玄上之音平台的密钥
   - **Base URL** — 接入点地址
4. 输入提示词、上传参考素材、调整生成参数
5. 点击「提交生成任务」，等待轮询完成并预览结果

## 文件托管方式

| 方式 | 特点 |
|---|---|
| **Litterbox** | 最大 1GB，可选 1h / 12h / 24h / 72h 过期 |
| **Uguu** | 最大 128MB，48h 过期 |
| **tflink** | 国内镜像，备用方案 |

> 本地文件上传后会按顺序尝试上述托管服务，获取公网 URL 后自动送入生成接口。

## 技术说明

- **无框架依赖**：原生 HTML + CSS + JavaScript，零 npm 包
- **数据持久化**：localStorage 保存配置、历史记录、系统日志、任务记录
- **素材引用**：提示词中的 `@类型#序号` 会在提交前解析为素材库 URL
- **签名计算**：素材库接口使用前端实时计算的 HMAC-SHA256 签名，无需后端

## 浏览器兼容性
