当前版本使用本地 Python 服务代理 Ark 请求。请先运行 python app.py，再通过 http://127.0.0.1:18765 打开页面；默认端口使用较少见的 18765，尽量避开常见本地服务冲突。 旧版素材库曾尝试直连火山引擎 AK/SK，当前版本已不再使用该方式。
当前版本改为浏览器直连，不再依赖本地 Python 代理服务。Seedance 生成接口直接请求 ark.cn-beijing.volces.com， 字幕擦除直接请求 amk.cn-beijing.volces.com，素材库接口走平台网关 /v1/ark-assets/*， 使用平台文档里的 Authorization: Bearer <sk-...> 鉴权方式。 如果浏览器报网络错误，通常是目标站点未放开 CORS、被浏览器插件拦截，或当前网络环境无法直连。
